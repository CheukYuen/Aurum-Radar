# 一、整体设计要求

请保持 4 张截图中的统一视觉风格，不要重新发明一套完全不同的 UI。

整体风格关键词：

```text
白金色
香槟金
珍珠白
浅米色
柔和阴影
高级珠宝质感
克制奢华
企业级专业感
MVP 简洁可用
```

设计应体现珠宝行业高级感，但不能像营销海报，也不能太浮夸。它应该是一个企业内部使用的专业情报系统。

视觉要求：

```text
1. 使用浅色系，不要黑金风格。
2. 主背景以珍珠白、象牙白、浅米色为主。
3. 强调色使用香槟金、柔和金色。
4. 正向状态使用柔和绿色。
5. 风险状态使用柔和红色。
6. 所有卡片使用圆角、轻描边、柔和投影。
7. 页面需要留白充足，信息层级清晰。
8. 保留少量钻石、宝石切面、丝绸、金属描边等珠宝行业装饰细节。
9. 不要使用任何真实珠宝品牌 Logo。
10. 不要加入搜索框，MVP 阶段暂不做搜索功能。
```

# 三、四个页面的产品定位

请务必让四个页面的信息层级不同，避免重复。

## 1. 概览页定位

概览页回答的问题是：

```text
全球这么多市场里，哪个国家值得关注？
```

信息粒度：国家级。

概览页应展示：

```text
1. 世界地图
2. 国家机会节点
3. 当前选中国家的国家级摘要
4. 关键分析摘要
5. 业务影响与建议
```

右侧国家摘要面板可以展示：

```text
新加坡
机会高

综合指数：82/100
政策环境：友好
市场增速：+12%
竞争强度：中高

市场亮点：
- 高净值消费稳定增长
- 旅游零售恢复
- 高端珠宝需求旺盛
- 电商与线下协同成熟
```

概览页只做国家级判断，不要展示具体商圈、门店数量、区域运营风险。

## 2. 地图洞察页定位

地图洞察页回答的问题是：

```text
这个国家内部，哪个城市 / 商圈 / 区域值得重点布局？
```

信息粒度：区域 / 商圈级。

注意：地图洞察页不要再重复概览页的“新加坡综合指数、市场亮点”等国家级内容。

地图洞察页应展示选定国家内部地图，例如新加坡地图。

进入状态：

```text
当前国家：新加坡
当前选中区域：乌节路
```

主地图要求：

```text
1. 展示新加坡本地地图，而不是世界地图。
2. 展示不同区域 / 商圈节点。
3. 默认高亮乌节路。
4. 节点展示周大福门店数量。
```

示例节点：

```text
乌节路：5 家门店
滨海湾：3 家门店
武吉士：3 家门店
裕廊东：2 家门店
淡滨尼：2 家门店
樟宜：1 家门店
市中心：3 家门店
```

地图交互提示：

```text
点击区域节点 → 右侧查看该商圈详细洞察与机会建议
```

右侧面板应是区域详情，不是国家摘要。

示例右侧标题：

```text
乌节路商圈
高优先级区域
区域详情
```

右侧指标：

```text
门店数量：5 家
商圈热力：高
竞品密度：高
客流结构：游客 + 高净值本地客群
```

区域画像：

```text
区域类型：高端商业核心区
主要场景：奢侈品购物中心、婚庆珠宝、高端礼赠
消费特征：品牌导向强，客单价高
适合动作：品牌曝光、会员活动、节日营销
```

门店洞察：

```text
周大福门店密度最高
周边高端珠宝品牌集中
适合品牌曝光与会员活动
租金成本较高，需关注转化效率
```

建议动作：

```text
加强核心商场陈列曝光
策划高端礼赠主题活动
监测 Tiffany / Cartier / 周生生竞品动作
```

底部模块改为区域运营分析，不要出现“区域对比”和“国家清单”。

底部模块建议：

```text
1. 门店分布概览
2. 核心商圈热力
3. 客流与消费信号
4. 区域运营风险
```

## 3. 情报中心页定位

情报中心页回答的问题是：

```text
最近发生了什么重要事件？这些事件对市场和业务有什么影响？
```

信息粒度：事件级。

页面结构：

```text
1. 顶部统计卡片
2. 情报分类标签
3. 左侧情报事件列表
4. 右侧事件详情面板
```

顶部统计：

```text
今日新增情报：28 条
高优先级事件：7 条
待跟踪事项：15 项
```

分类标签：

```text
全部
竞争
产品
社媒
法规
渠道
```

情报列表每条卡片包含：

```text
分类标签
事件标题
事件摘要
来源
发布时间
点击入口
```

右侧事件详情包含：

```text
事件摘要
潜在影响
关联市场
相关品牌 / 平台
来源引用
```

来源引用必须显著保留，因为系统需要强调情报可追溯。

示例事件：

```text
竞争
LVMH 收购意大利高级珠宝品牌 Damiani 多数股权
来源：行业报告 / Bain & Company
时间：05/31 09:30
```

详情示例：

```text
事件摘要：
LVMH 集团已签署协议收购意大利高级珠宝品牌 Damiani 的多数股权。

潜在影响：
- 竞争格局：高端珠宝市场竞争加剧
- 品牌策略：区域品牌可能获得全球资源支持
- 市场趋势：意大利工艺与设计价值被进一步强化

关联市场：
欧洲 / 北美 / 中东 / 亚太高端市场

相关品牌：
LVMH / Damiani / Tiffany & Co. / Cartier

来源引用：
Bain & Company – Luxury Goods Worldwide Market Study 2024
```

# 五、MVP 边界

请围绕 MVP 设计，不要增加太复杂的能力。

MVP 保留：

```text
1. 全球概览地图
2. 国家级机会摘要
3. 国家内部地图洞察
4. 区域 / 商圈详情
5. 情报事件列表
6. 情报详情与来源引用
7. 部门行动建议
8. 行动清单详情
9. 简单筛选：时间、地区、品类
```

MVP 暂不做：

```text
1. 全局搜索
2. 复杂权限系统
3. 实时协同编辑
4. 多级审批流
5. 自动任务派发
6. 深度 BI 自定义报表
7. 复杂图表配置
8. 数据源管理后台
```

请严格遵守以下原则：
